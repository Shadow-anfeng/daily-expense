# 每日花销

这是一个可以发布到 GitHub Pages 的个人记账工具。

## 发布到 GitHub Pages

1. 在 GitHub 新建一个仓库，例如 `daily-expense`。
2. 把本文件夹里的所有文件上传到仓库根目录。
3. 打开仓库的 `Settings`。
4. 进入 `Pages`。
5. `Build and deployment` 选择 `Deploy from a branch`。
6. `Branch` 选择 `main`，目录选择 `/root`。
7. 保存后等待一会儿，GitHub 会生成一个网址。

## 手机使用

打开 GitHub Pages 网址后：

- iPhone：Safari 分享按钮 -> 添加到主屏幕。
- Android：Chrome 菜单 -> 添加到主屏幕。

## 数据说明

当前版本的数据保存在当前设备的浏览器里。不同手机和电脑之间不会自动同步。

换设备时可以：

1. 在旧设备点击“导出数据”。
2. 把 JSON 文件发到新设备。
3. 在新设备点击“批量导入”里的“恢复备份”。
